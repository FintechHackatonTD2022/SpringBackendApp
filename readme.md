**gimmebackend** is a Spring Backend application made for Case #3 of [Fintech Hackathon, hosted in Terricon Valley](https://terricon.kz/ru/hackathon)  
Used as a server-side application

## Installation

1. Clone this repository
2. Run server with:
   `gradlew bootRun`
